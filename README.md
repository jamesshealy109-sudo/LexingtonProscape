# Lexington Proscape website

This is a static, mobile-friendly single-page website built for GitHub Pages or any basic web host.

## Files

- `index.html` — full site, including CSS and JavaScript
- `assets/lexington-proscape-logo.jpg` — client logo
- `assets/strataworks-logo.png` — linked footer credit
- `assets/favicon.png` — browser icon

## Before final launch

1. Add the client's phone number and business email.
2. Replace the Facebook-copy contact workflow with direct form delivery once the email is known. FormSubmit can be added without a server.
3. Add 4–8 strong project photos once available. A real project gallery will add more credibility than stock landscaping images.
4. Confirm the exact list of hardscape and maintenance services offered.
5. Add the final domain to Open Graph metadata and any structured local-business data.

## Current contact behavior

The estimate form formats the visitor's request, copies it to the clipboard, and opens the public Lexington Proscape Facebook page. This keeps the draft functional before a phone number or email address is supplied.

## Deployment

Upload the contents of this folder to the root of a GitHub Pages repository. Keep the `assets` folder next to `index.html`.
